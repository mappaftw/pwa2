# My App

Generated with Mappa AI App Builder.

## Getting Started

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
```

Deploy the `dist` folder to any static host (Vercel, Netlify, etc.).

## Install as App

This project is a **Progressive Web App**. After deploying to HTTPS:

- **Android**: Open in Chrome → tap **Install App**
- **iPhone**: Open in Safari → **Share** → **Add to Home Screen**
- **Desktop**: Chrome → address bar install icon

The app runs fullscreen with your icon, works offline, and gets
automatic updates — no app store needed.

## Customize

- **App name & colors**: `public/manifest.json`
- **Icons**: replace `public/icon-192.png` and `public/icon-512.png`
